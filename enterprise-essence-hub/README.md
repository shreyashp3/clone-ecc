# Enterprise Essence Hub

Enterprise cloud solutions marketing platform for ECC Technologies.

## Project Setup

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Quick Start

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and instant preview.
npm run dev
```

### Development

- **Local IDE**: Edit files in your preferred code editor and push changes
- **GitHub Direct**: Navigate to files, edit directly in GitHub, and commit
- **GitHub Codespaces**: Use cloud-based development environment

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Node.js REST API

## Deployment

The application is deployed automatically on GitHub push to the main branch.
